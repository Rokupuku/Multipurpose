실습 예제 3,4

CSS 스타일 속성

CSS의 스타일 속성 중에서 색상을 값으로 사용하는 속성은 여러 개 있습니다.

color 속성도 색상을 값으로 사용하는 속성 중 하나입니다.
색상 이름을 사용하여 글자색을 변경할 수도 있습니다.

키워드 표기법

  1. 색상의 영문명을 속성값으로 사용
  2. color 속성의 red도 색상의 영문명 속성값
  3. 색상 키워드
  4. 종류_150개 이상
  5. 속성값 자동 완성 기능
